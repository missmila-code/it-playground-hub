# IT Playground Portal

Локален учебен портал за игри, тестове и практически мисии по Информационни технологии.

## Структура

- `index.html` - начална страница с вход към класовете.
- `grade-5.html` до `grade-10.html` - отделни страници по класове.
- `dashboard.html` - учителски център за линкове към приложения, таблици с резултати и бъдещи отчети.
- `styles.css` - визуален стил и responsive поведение.
- `script.js` - малко поведение за мобилното меню.

## Как се използва локално

Отвори `index.html` в браузър. Това е статичен сайт и не изисква сървър за базов преглед.

## Каква е ролята на dashboard страницата

`dashboard.html` е място за учителски линкове и организация:

- линкове към готови Apps Script игри;
- линкове към таблици с резултати, добавени ръчно от учителя;
- списък с планирани разработки;
- бъдещи отчети или инструкции.

Не съхранявай реални ученически лични данни в локалните файлове.

## Бъдещ Vercel deploy

Когато порталът е готов:

1. Създай GitHub repository.
2. Качи тази папка като проект.
3. Във Vercel избери Add New Project и Import Git Repository.
4. Избери repository-то.
5. Натисни Deploy.

Не се деплойва автоматично от този локален workspace.
